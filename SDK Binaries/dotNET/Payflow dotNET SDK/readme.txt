The SDK was built using .NET 5; however, the samples are set for .NET 4.8.

To use this SDK, open the Solution in Visual Studio 2019/2022. View the default sample called DoSaleComplete.cs under Payflow dotNET SDK\SamplesCS\src\PayPal\Payments\Samples\CS\DataObjects\BasicTransactions.

Update the user credentials and give it a run.

See the API documention under Docs >> Help, open index.html.

Also, review the Payflow Developer Guide at https://developer.paypal.com/api/nvp-soap/payflow/ for more information around the parameters.
